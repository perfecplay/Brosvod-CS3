# ☁️ Brosvod CS3 — CloudStream Türkçe Eklentiler

CloudStream için Türkçe içerik sağlayıcılarını `.cs3` biçiminde derleyip yayımlayan kaynak deposudur.

## 💾 Kurulum

CloudStream depo adresi:

```text
https://raw.githubusercontent.com/perfecplay/Brosvod-CS3/refs/heads/builds/repo.json
```

CloudStream protokol adresi:

```text
cloudstreamrepo://raw.githubusercontent.com/perfecplay/Brosvod-CS3/refs/heads/builds/repo.json
```

> `builds` dalı oluşturulup ilk GitHub Actions derlemesi tamamlandıktan sonra bu adres çalışır.

## 🛠️ Derleme

Kaynak dalı `main`, derlenmiş `.cs3` paketlerinin yayın dalı `builds` olarak yapılandırılmıştır. GitHub Actions, kaynak değişikliklerinden sonra eklentileri otomatik derler ve `plugins.json` ile birlikte `builds` dalına gönderir.

## 🙏 Kaynak ve katkılar

Eklenti kodlarındaki orijinal geliştirici ve katkı bilgileri korunmuştur. Bu depo, kaynakların `perfecplay/Brosvod-CS3` altında derlenip yayımlanması için yapılandırılmıştır.
